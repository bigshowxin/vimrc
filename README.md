#我的vimrc配置

----

**我的插件**

    Vundle
    NEADTree
    vim-Rails
    delimitmate
    vim-markdown
    ctrl-p


**使用方法**

- 下载到本地


    git clone git@github.com:guochy2012/vimrc.git

- 复制

    cp vimrc/vimrc ~/.vimrc
    
- 安装Vundle

    [Vundle](https://github.com/gmarik/Vundle.vim "")

- 安装其余插件

    vim +BundleInstall
